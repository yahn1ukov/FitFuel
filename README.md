# FitFuel

## About

### Description

FitFuel is a pet-project that provides the ability to find out interesting recipes

### Architecture

Clean Architecture with MVVM Pattern

### Technologies

- Kotlin (with Coroutines)
- Room
- Retrofit
- OkHttp
- Gson
- Navigation
- Dagger-Hilt
- LiveData
- Coil
- DiffUtil
- DataStore
- DataBinding

### Functionality

- Recipe:
    - Local (caching):
        - Get all
        - Insert
    - Remote:
        - Get all
- Favorite recipes:
    - Get all
    - Insert
    - Delete
    - Delete all
